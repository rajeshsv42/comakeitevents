HTML is written in the form of HTML elements consisting of tags enclosed in angle brackets , within the web page content. HTML tags most commonly come in pairs like and although some tags represent empty elements and so are unpaired. The first tag in a pair is the start tag, and the second tag is the end tag (they are also called opening tags and closing tags). In between these tags web designers can add text, further tags, comments and other types of text-based content.
